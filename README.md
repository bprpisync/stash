# stash-randomBackgrounds
Changes the stash background to a random image of your choosing


Add 'background' tag to images you want to use. The script automatically changes your background each page you visit.
It uses GraphQl to fetch images on your stash with the 'background' tag. But it is smart and will look at the page you're on.
So it will look for specific images from specific performers, studio's, groups and more.

How to use? Install it using the index.yml above and add 'background' tags to (a minimum of 2) images of your choosing.
